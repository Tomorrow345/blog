JS中的随机数

  • 编程
当你在做猜数字、掷骰子等一类网页时，都需要用js生成随机数，我们可以通过Math.random来实现
JS 
                            
// 生成1-100的随机数
let rollNum = Math.floor(Math.random() * (100 - 1 + 1)) + 1;
用let创建一个变量，Math.floor用来将数字转化为整数，如果没有结果可能会是：3.0451182108912738、6.923154507599182、3.5553578526501806…… Math.random() * (最大值 - 最小值 + 最小值) + 最小值 下面是关于本网站掷骰子的代码
JS 
                            
const rollBtn = document.getElementById('rollBtn'); 
const points = document.getElementById('points'); 

rollBtn.onclick = () => {
    let rollNum = Math.floor(Math.random() * (6 - 1 + 1)) + 1; 
    points.textContent = `点数：${rollNum.toString()}`; 
}
找到掷骰子的按钮和显示点数的文字，给按钮添加点击事件，创建一个随机数变量，并修改文字的内容
