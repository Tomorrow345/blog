想让网页的文字看起来不那么生硬，可以在css中使用font-family修改字体，需要自己寻找好看的字体，自己电脑有的其它电脑可能没有，就会变成默认的，可以尽量寻找更兼容的字体，如果浏览网页的电脑是清朝下来的，什么字体都没有，就需要在后面加上sans-serif，sans-serif会让电脑寻找一个等线字体，每个系统都有自己的等线字体
CSS 

.item {
    font-family: 'Microsoft YaHei', Arial, sans-serif
}
